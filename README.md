## 🧠 What is ODI AI Team Selector?

An intelligent ODI cricket team selector that recommends the optimal playing XI using real player statistics, venue conditions, pitch types and ML-based scoring algorithms. Supports 8 countries and 40+ international venues with detailed batting, bowling and strategy insights.

---

## 🚀 Features

| Feature | Description |
|---------|-------------|
| 🏟️ 40+ Venues | Detailed pitch profiles for grounds across 8 countries |
| 🎯 Smart Selection | Pitch-aware ML scoring for batters and bowlers |
| 👁️ Player Profiles | Full career stats with performance index bars |
| 📊 Analytics Dashboard | Radar, donut, bar charts powered by Plotly |
| 🌀 Pitch Override | Auto-detect or manually set pace / spin / balanced |
| 💧 Dew Factor | Toss strategy recommendations based on dew data |
| 🔁 All-Rounder Balance | Role-balanced XI with WK, AR, bowler slots |
| 🏴 8 Countries | India, Australia, England, Pakistan, Bangladesh, SA, SL, NZ |
| 🎨 Country Themes | Dynamic UI theme changes per selected country |
| ⚡ FastAPI Backend | REST API endpoint for Best XI integration |

## ⚙️ Setup & Run Locally

**1. Clone the repo**
```bash
git clone https://github.com/24msrds/Final-ODI-selector.git
cd Final-ODI-selector
```

**2. Install dependencies**
```bash
pip install -r requirements.txt
```

**3. Run Streamlit app**
```bash
streamlit run frontend/app.py
```
